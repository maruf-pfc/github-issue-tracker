# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | ✅                 |
| < v1.x  | ❌                 |

## Reporting a Vulnerability

We take security seriously. If you discover any security-related issues, please help us keep the community safe by reporting them responsibly.

**Please do not open a public GitHub issue.** Instead, send an email to:

`developers@blackstart-labs.com`

Include as much information as possible, such as:
- Description of the vulnerability.
- Steps to reproduce.
- Potential impact.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.
